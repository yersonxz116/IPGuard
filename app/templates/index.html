Misión y Visión

Seguridad inteligente para hogares conectados

01

Misión

Proporcionar una solución de seguridad accesible y eficiente que permita a los usuarios monitorear sus espacios en tiempo real mediante cámaras IP, inteligencia artificial y notificaciones instantáneas que garanticen la protección del hogar y sus seres queridos.

02

Visión

Convertirse en una referencia nacional en sistemas de vigilancia inteligente doméstica, innovando continuamente en la detección de objetos y análisis de video mediante redes neuronales profundas, democratizando el acceso a tecnología de seguridad avanzada para todos los hogares.

Objetivos

Ruta de desarrollo del sistema de vigilancia inteligente

Objetivo General

Implementar un sistema de monitoreo y detección de personas en tiempo real para cámaras IP, utilizando MediaPipe Pose y OpenCV mediante un servidor Flask con interfaz web intuitiva y notificaciones automáticas de WhatsApp.

Objetivos Específicos

1

Analizar los requerimientos del sistema de monitoreo y las técnicas de detección de personas basadas en MediaPipe y OpenCV para determinar la configuración óptima del modelo Pose Landmarker.

2

Implementar un servidor Flask con API REST para la comunicación entre el módulo de detección y la interfaz web.

3

Implementar una interfaz web responsive que permita gestionar cámaras IP, visualizar detecciones y configurar horarios de monitoreo.

4

Implementar un sistema de notificaciones automáticas mediante WhatsApp que alerte al usuario cuando se detecte una persona.
