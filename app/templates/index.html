Misión y Visión

Misión

Proporcionar una solución de seguridad accesible y eficiente que permita a los usuarios monitorear sus espacios en tiempo real mediante cámaras IP, utilizando inteligencia artificial para la detección automática de personas y notificaciones instantáneas que garanticen la protección del hogar y sus seres queridos.

Visión

Convertirse en una referencia nacional en sistemas de vigilancia inteligente domestica, innovando continuamente en la detección de objetos y el análisis de video mediante redes neuronales profundas, democratizando el acceso a tecnología de seguridad avanzada para todos los hogares.

Objetivos

Objetivo General

Desarrollar un sistema de monitoreo y detección de personas en tiempo real para cámaras IP, utilizando una red neuronal convolucional basada en yolo11s con arquitectura Faster R-CNN, implementada mediante un servidor Flask con interfaz web intuitiva y notificaciones automáticas de WhatsApp.

Objetivos Específicos

  • Diseñar y entrenar un modelo de detección de objetos utilizando transfer learning con yolo11s como backbone.
  • Implementar un servidor Flask con API REST para la comunicación entre el modelo de IA y la interfaz web.
  • Desarrollar una interfaz web responsive que permita gestionar cámaras IP, visualizar detecciones y configurar horarios de monitoreo.
  • Integrar un sistema de notificaciones automáticas mediante WhatsApp que alerte al usuario cuando se detecte una persona.
